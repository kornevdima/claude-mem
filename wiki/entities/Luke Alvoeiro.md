# Luke Alvoeiro

Leads the core agent harness at [[Factory]]. Dev-tools background: ~2.5 years before the talk (i.e. circa 2023) he started a project at Block that evolved into **Goose**, now one of the leading open-source coding agents, donated to the Agentic AI Foundation.

Speaker of the AI Engineer 2026 talk "The Multi-Agent Architecture That Actually Ships" ([[yt-alvoeiro-multi-agent-architecture]]), where he proposed the five-primitive [[Multi-Agent Communication Taxonomy]] and presented Factory's missions architecture ([[Validation Contract]], [[Structured Handoff]], serial execution with read-only parallelism, per-role model selection a.k.a. "droid whispering").

## Key positions

- "The bottleneck in software engineering nowadays is not intelligence. It's limited by human attention."
- "Tests written after implementation don't catch bugs. They confirm decisions."
- Architectures should be bitter-lesson-proof: orchestration in prompts/skills so the system improves with every model release.
