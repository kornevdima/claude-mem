# Matt Pocock

Developer educator, formerly known for Total TypeScript; now teaches AI-assisted development (site: AI Hero, plus a Claude Code course). Delivered the 2026-04-24 AI Engineer workshop ingested as [[yt-pocock-ai-coding-workflow]].

## Positions

- Software engineering fundamentals (Brooks, Ousterhout, Pragmatic Programmer) transfer directly to agent workflows; old software books are "an absolute gold mine" for prompts.
- Anti specs-to-code: "the code is your battleground" — keep a handle on it throughout.
- Prefers clearing context over compaction; treats ~100K tokens as the smart-zone boundary; runs a token status line always.
- Own your planning stack rather than adopting frameworks (Spec Kit, OpenSpec, Taskmaster) — inversion of control.
- Uses Claude Code but is vendor-skeptical ("democracy is the worst way to run a country apart from all the other ways"); barely uses CLAUDE.md; prefers pull-based skills.
